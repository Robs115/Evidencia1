Descripcion de etiquetas HTML:

<!DOCTYPE>:

Tecnicamente no es una etiqueta, es informacion que se le da al navegador para que sepa que tipo de documento esta tratando.

<html>:

Representa la raiz de un documento HTML, dentro de esta etiqueta se encuentra todos los otros elementos html (salvo el DOCTYPE). Se recomienda siempre indicar el atributo de idioma (ej. lang="es").

<head>:

Es un contenedor de metadatos (datos sobre datos), se coloca entre el html y el body. Esta informacion es acerca del documento HTML, no se despliega en la pagina, por ejemplo, el titulo del documento, scripts, estilos, etc.

<title>:

Define el titulo del documento, es mostrado en la pestaña de pagina del navegador. El titulo de la pagina es usado por algoritmos de motores de busqueda para mostrar resultados.

<meta>:

Especifica el tipo de caracteres, describe la pagina, el autor, etc. No se muestra en la pagina, pero es informacion usada por el navegador, buscadores y otros servicios.

<body>:

Define el cuerpo del documento, contiene multitud de elementos HTML, como encabezados, parrafos, imagenes, vinculos, etc.

<h1>, <h2>, <h3>, <h4>, <h5>, <h6>:

Se usan para definir encabezados, del mas grande al mas chico, siendo h1 el mas grande y h6 el mas chico.

<p>:

Se usa para definir parrafos de texto. Por predeterminado los navegadores añaden un espacio en blanco entre cada p.

<u>:

Se usa para subrayar un texto.

<b> y <strong>:

Define texto en negritas, b se refiere a texto sin importancia especial, strong se usa para texto importante.

<i>:

Define texto en cursiva o con tono alternativo.

<hr>:

Define un separador de tema o contenido, tipicamente se muestra como una linea horizontal. Es una etiqueta que no requiere ser cerrada.

<br>:

Define un salto de linea, un renglon en blanco. Es una etiqueta que no requiere ser cerrada.

<ul>:

Define una lista desordenada.

<ol>:

Define una lista ordenada. Con atributos se puede hacer que use letras, numeros, etc.

<li>:

Un objeto de una lista, se usan dentro de una lista ordenada o desordenada.

<a>:

Define un hipervinculo, para enlazar una pagina a otra.

<mark>:

Define texto resaltado. Lo resalta con color amarillo.

<small>:

Define texto chico, se suele usar para cosas como comentarios laterales o copyright.

<em>:

Le da enfasis al texto, parecido a <i>

<cite>:

Cita el titulo de un trabajo creativo (ej. un libro, poema, cancion, pelicula, etc.)

<q>:

Define que un fragmento o frase breve en un texto es una cita textual, los navegadores lo suelen mostrar entre comillas. Para citas largas se usa blockquote

<blockquote>:

Define un bloque de texto con una cita textual.

<abbr>:

Define una abreviacion o acronimo, se usa el atributo titulo para mostrar la descripcion de la abreviacion/acronimo cuando colocas el puntero del raton sobre el elemento.

<sup>:

Define que el texto se muestre como superíndice.

<sub>:

Define que el texto se muestre como subíndice.

<del>:

Define texto eliminado del documento, tipicamente se muestra como texto con una linea atravesandolo por en medio.

<img>:

Se usa para poner una imagen en una pagina HTML, se requiere usar el atributo src para la fuente de la imagen y alt para mostrar un texto descriptivo en caso de que no se halle la imagen.

<iframe>:

Se usa para mostrar una pagina web dentro de una pagine web.

<code>:

Se usa para mostrar texto que representa codigo de computadora, usa un tipo de fuente particular.

<video>:

Se usa para mostrar un video en HTML. El texto entre las etiquetas video solo se muestra si el navegador no tiene soporte para esta etiqueta.
