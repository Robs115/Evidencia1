!DOCTYPE

Es una declaración que se coloca al principio de un documento HTML y que indica al navegador qué versión de HTML se está utilizando


HTML

Representa la raíz de un documento HTML y engloba todo el contenido de la página web


head

Es un contenedor que almacena metadatos, es decir, información sobre la página web, pero no se muestra directamente en el contenido visible del navegador


meta

Se utiliza para proporcionar metadatos sobre una página web.


title

Se utiliza para definir el título de un documento web.


body

Representa todo el contenido que se muestra al usuario en el navegador.


h1 -> h6

Define el encabezado principal o título de una página web. Teniendo diferentes tamaños desde el h1 mas grande al h6 mas pequeño.


p

Se utiliza para definir un párrafo de texto. Ejemplo: <p>Lorem, ipsum dolor.</p>.


i

Se utiliza para marcar una sección de texto con una voz o estilo alternativo, generalmente mostrando el texto en cursiva. Ejemplo: <p><i>Lorem</i>, ipsum dolor.</p>


hr

Representa una regla horizontal, que se utiliza para crear una línea divisoria horizontal en una página web Ejemplo:


br

Se utiliza para insertar un salto de línea en el texto Ejemplo


Fin Ejemplo


u

Se utiliza para definir texto subrayado. Ejemplo:<p><u>Lorem</u>, ipsum dolor.</p>


Strong

Se utiliza para resaltar un texto que es importante o debe ser resaltado con énfasis Ejemplo:<p><strong>Lorem</strong>, ipsum dolor.</p>


li

Define un elemento de una lista. Se utiliza dentro de <ul> o <ol>.

Ejemplo:

  • Manzana
  • Pera

ul

Define una lista desordenada, cuyos elementos aparecen con viñetas.

Ejemplo:

  • Rojo
  • Verde


ol

Define una lista ordenada, cuyos elementos aparecen numerados.

Ejemplo:

  1. Primero
  2. Segundo


mark

Resalta texto, como si estuviera marcado con un resaltador.

Ejemplo: <p>Este es un texto <mark>importante</mark></p>


small

Muestra el texto en un tamaño más pequeño.

Ejemplo: <p>Texto normal <small>y este más pequeño</small></p>


em

Resalta un texto con énfasis, mostrándolo en cursiva.

Ejemplo: <p>Esto es <em>muy importante</em></p>


cite

Se usa para citar el título de una obra, libro, película, etc.

Ejemplo: <p>Leí <cite>Don Quijote</cite> el año pasado.</p>


q

Define una cita corta en línea (entre comillas).

Ejemplo: <p>Ella dijo: <q>El conocimiento es poder</q></p>


blockquote

Se utiliza para citas largas en bloque.

Ejemplo:

<blockquote>La educación es el arma más poderosa para cambiar el mundo.</blockquote>


abbr

Define una abreviatura, mostrando el significado al pasar el cursor.

Ejemplo: <p><abbr title="Organización de las Naciones Unidas">ONU</abbr></p>


code

Representa fragmentos de código de computadora.

Ejemplo: <p>El comando es <code>console.log("Hola")</code> </p>


Superindice

Se utiliza la etiqueta <sup> para mostrar texto en superíndice.

Ejemplo:E = mc2


Subindice

Se utiliza la etiqueta <sub> para mostrar texto en subíndice.

Ejemplo: H2O


del

Marca texto como eliminado o tachado.

Ejemplo: <p>Precio: <del>$50</del> $30</p>


a

Define un hipervínculo que lleva a otra página o recurso.

Ejemplo: <a href="https://www.google.com" >Ir a Google</a>
