Evidencia 1 : Referencias visuales de tags HTML.

<h1> a <h6>

Uso: Se utilizan para definir encabezados, siendo <h1> el más importante y <h6> el menos.

<h1>Encabezado 1</h1>
<h3>Encabezado 3</h3>

Encabezado 1

Encabezado 3

<p>

Uso: Define un párrafo de texto.

<p>Este es un párrafo.</p>

Este es un párrafo.

<hr>

Uso: Inserta una línea horizontal.

<hr>

<br>

Uso: Salto de línea dentro del texto.

Hola<br>Mundo
Hola
Mundo

<strong> / <b>

Uso: Ambos ponen el texto en negritas. <strong> también indica importancia semántica.

<strong>Importante</strong> y <b>Negrita</b>
Importante y Negrita

<mark>

Uso: Resalta texto como si fuera un marcador.

<mark>Texto resaltado</mark>
Texto resaltado

<small>

Uso: Muestra texto en tamaño más pequeño.

<small>Nota pequeña</small>
Nota pequeña

<em>

Uso: Da énfasis al texto (cursiva con valor semántico).

<em>Texto enfatizado</em>
Texto enfatizado

<blockquote>

Uso: Se usa para citas en bloque.

<blockquote>Esta es una cita.</blockquote>
Esta es una cita.

<abbr>

Uso: Define una abreviatura, mostrando el significado al pasar el cursor.

<abbr title="HyperText Markup Language">HTML</abbr>
HTML

<code>

Uso: Representa código de computadora.

<code>console.log("Hola")</code>
console.log("Hola")

<del>

Uso: Texto eliminado o tachado.

<del>Texto eliminado</del>
Texto eliminado

<i>

Uso: Texto en cursiva.

<i>Texto en cursiva</i>
Texto en cursiva

<u>

Uso: Texto subrayado.

<u>Texto subrayado</u>
Texto subrayado

<ol>, <ul>, <li>

Uso: Listas ordenadas (<ol>) y no ordenadas (<ul>) con elementos (<li>).

<ol>
  <li>Primero</li>
  <li>Segundo</li>
</ol>

<ul>
  <li>Manzana</li>
  <li>Banana</li>
</ul>
  1. Primero
  2. Segundo
  • Manzana
  • Banana

<img>

Uso: Inserta una imagen.

<img src="https://i.imgur.com/8c3NaCB.jpeg" alt="Ejemplo">

<iframe>

Uso: Inserta otra página web o contenido externo.

<iframe src= https://wplace.live/ ></iframe>

<video>

Uso: Inserta un video con controles.

<video width="320" controls>
  <source src="https://www.youtube.com/watch?v=5NIA8Ef5C-Q" type="https://www.youtube.com/watch?v=5NIA8Ef5C-Q">
  
</video>

Uso: Muestra información expandible al dar clic.

<details>
  <summary>Ver más</summary>
  Texto oculto que se despliega.
