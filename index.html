Bienvenido

El desarrollo web es una de las áreas más dinámicas y en constante evolución dentro de la tecnología. Aquí aprenderás los elementos fundamentales para crear páginas y aplicaciones que funcionen en cualquier navegador moderno.

Introducción al Desarrollo Web

El desarrollo web combina distintas tecnologías que permiten dar vida a sitios y aplicaciones en internet. Sus características principales son:

  • Interactividad: comunicación entre usuarios y sistemas.
  • Accesibilidad: disponible para cualquier persona con internet.
  • Escalabilidad: los proyectos pueden crecer según la necesidad.
  • Multiplataforma: funciona en diferentes dispositivos.

Lenguaje HTML

HTML (HyperText Markup Language) es el lenguaje que define la estructura de una página web. Con él se organizan títulos, párrafos, listas, enlaces, imágenes y más. HTML es un leguaje descriptivo, no de programacion, y funciona con diseños gracias a css, y la interactividad de la pagina es añadida con JavaScript.

Estructura

En HTML existen etiquetas muy utilizadas para dar formato al texto: los títulos que van desde <h1> hasta <h6>, los párrafos con <p>, saltos de línea con <br>, líneas divisorias con <hr>, además de las etiquetas de énfasis como <b> o <strong> para resaltar en negritas, <i> o <em> para cursivas, y <u> para subrayados. También se han incorporado otras etiquetas modernas como <mark> para resaltar texto, <small> para mostrar notas en un tamaño menor, o <sub> y <sup> que permiten escribir subíndices y superíndices.

Otro elemento fundamental son las listas, que permiten organizar información. Las listas ordenadas <ol> presentan los elementos de forma numerada, mientras que las listas no ordenadas <ul> muestran viñetas simples. En ambos casos cada elemento se define con <li>. Asimismo, HTML permite añadir enlaces mediante <a> y trabajar con contenido multimedia como imágenes con <img>, videos con <video> o incluso mapas y contenido externo con <iframe>.

Para organizar datos de manera estructurada también se emplean las tablas, creadas con <table>. Estas se dividen en filas <tr> y celdas <td>, mientras que los encabezados se representan con <th>. Además, se pueden añadir etiquetas semánticas como <thead>, <tbody> o <tfoot> para diferenciar claramente las secciones de la tabla.

Lenguaje css

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación y el diseño visual de un documento escrito en un lenguaje de marcado como HTML. Es una herramienta esencial en el desarrollo web, ya que permite separar el contenido (HTML) de su apariencia, facilitando la personalización y el mantenimiento de los sitios web.

Propiedades más importantes

Texto y fuentes

  • color: color del texto
  • font-family: tipo de letra
  • font-size: tamaño de letra.
  • text-align: alineación (left, center, right, justify)
  • text-decoration: subrayado, tachado, etc

Modelo de caja

  • width / height: ancho y alto
  • margin: espacio exterior
  • padding: espacio interior
  • border: borde (grosor, estilo y color)

Fondo y colores

  • background-color: de fondo
  • background-size: tamaño de la imagen (cover, contain, etc.)
  • background-image: imagen de fondo

Flexbox

  • display: flex: activa flexbox.
  • justify-content: alinear horizontal (start, center, space-between, etc.)
  • align-items: alinear vertical (start, center, stretch…)
  • flex-direction: dirección (row, column).
  • flex-wrap: si los elementos se ajustan o no en varias líneas

Diseño y organización

  • display: tipo de caja (block, inline, flex, grid, none…)
  • position: posicionamiento (static, relative, absolute, fixed, sticky)
  • top, right, bottom, left: posición en la pantalla
  • float: flotar elementos (casi en desuso, se usa más flex o grid)

Otros muy usados

  • overflow: qué pasa si el contenido se sale (hidden, scroll, auto)
  • opacity: transparencia
  • cursor: tipo de cursor (pointer, default, etc.)
  • transition: animaciones suaves entre cambios
  • animation: animaciones avanzadas con @keyframes

Aplicaciones para programar:

Bloc de notas:

El Bloc de notas es una herramienta muy básica que viene incluida en Windows. Sirve para escribir y editar texto sin formato sin distracciones ni funciones avanzadas. Aunque no está pensado específicamente para desarrollo web, puede usarse para crear o modificar archivos HTML, CSS o JavaScript de manera sencilla. Su ventaja principal es que es ligero, rápido y siempre disponible, pero carece de ayudas como autocompletado, resaltado de sintaxis o extensiones. Es una opción accesible para ediciones rápidas o para aprender los fundamentos del código sin distracciones.

Visual Studio Code:

Uno de los mas usados, creado por Microsoft, es una herramienta creada especialmente para programar (en multitud de lenguajes), ligera, gratuita y multiplataforma, pero al mismo tiempo muy potente. Ofrece resaltado de sintaxis, autocompletado inteligente, depuración integrada, y una enorme variedad de extensiones que permiten trabajar con HTML, CSS, JavaScript, frameworks modernos (React, Angular, Vue), control de versiones con Git y hasta servidores locales. Además, su interfaz es sencilla y personalizable, lo que lo hace ideal tanto para principiantes como para desarrolladores avanzados.

Notepad++:

Notepad++ es un editor de texto ligero y gratuito para Windows, ideal para programadores y principiantes. Soporta múltiples lenguajes de programación, incluyendo HTML, CSS y JavaScript, con resaltado de sintaxis, autocompletado básico y pestañas para trabajar con varios archivos a la vez. Su principal ventaja es que es rápido, sencillo y muy accesible, perfecto para ediciones rápidas o aprender a codificar sin distracciones. Además, cuenta con plugins que amplían su funcionalidad según tus necesidades.
